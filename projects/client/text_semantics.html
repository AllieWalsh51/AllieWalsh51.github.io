The Basics

Links

Links (<a>) are what you use to get the reader to another webpage. They can be internal, and link to another location on the webpage, or external, taking the user to a different page. Inside the <a> element, the href attribute can be used to specify what the element is linked to.

Emphasis/Italics

Emphasis (<em>) is used to specify that text should be emphasized, and is usually represented as italics.

Importance/Boldface

Importance (<strong>) is what you use to specify that text is important, and is usually represented by boldface in the user's browser.
