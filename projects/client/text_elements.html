Text Content

Headings

Section headings are used to label content. There are six levels of headings, ranging from <h1> to <h6>.

Paragraphs

Paragraphs (<p>) are used for the majority of text, like what I'm doing here.

Blockquotes

Blockquotes are used when you're quoting a large chuck of text. You can use the <cite> attribute to cite the quote.

Lists

Ordered Lists

Ordered lists (<ol>) are used to list items with numbers.

Unordered Lists

Unordered lists (<ul>) are used to list items with bullet points.

List Elements

List elements (<li>) are used to specify an item within a list.

Definition Lists

Definition lists (<dl>) are used to list and define terms.

Definition Terms

Definition terms (<dt>) are the terms you are defining in a definition list.

Definition Description

The definition description is the definition of the term you are defining in the description list.

NOTE - Nesting Lists

Lists can be nested within each other, making a list inside a list.
