Terminal Tutorial!

Welcome to the terminal tutorial, a spot for your basic terminal commands free of charge.

Basic Commands

These commands are the basic commands you'll see in your life with the terminal.

cd
The command that moves you between directories: cd place-to-move.
ls
The command that shows you what's in a directory: ls.
pwd
The command that shows you what directory you're in: pwd.
mkdir
The command that creates a directory: mkdir directory-name.
rm
Removes files and directories you don't need: rm file-name.
cp
The command that copies files between directories: cp original-file copy-file.
mv
Move or rename files: mv file place-to-move.
grep
The command to search for specific text within files: grep text-to-seach place-to-search.
less
The command to interact with files: less file-name.

More useful info

  1. You use Ctrl + C to terminate a process.
  2. Use the sudo command (not listed) with major caution. It can really break stuff if used wrong.
  3. Use cd .. to backtrack to the last file you were at and use cd ~ to get back to your home directory

Vim!

Vim is a file editor within the terminal that I often use. To get into vim, simply type vim file-name. You move using the arrow keys, and in order to edit the file you must go into insert mode by pressing i. There are plenty of commands. Here's some html I wrote in vim. 

<body>
		<h1>Hello, world!</h1>
		<body>

Guess what it will say.
Hello, world!
