# THe TCP/IP Model

## Application

What does?

- Converts everything to binary
- Compresses it
- Interacts with you
- Many protocols 
	- WWW
	- IRC
	- SMS
	- VOIP
	- Many email protocols
	- Others

## Transport

- Converts info into packets
	- RCP (get all info)
	- UDP (one shot info)
- Gets it to internet

## Internet

- Gives computers IP addresses
- Routers work on this layer

## Link

- Transfers info to hard signals

- Protocols include:
	- Wifi
	- Ethernet
	- LTE
	- Bluetooth
- Transfers info from one device to the next
